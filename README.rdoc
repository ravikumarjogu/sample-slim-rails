== Sample Rails app with Slim

A sample rails app, demonstrate Slim template wriring with an bootstrap free template.

Its an alternate to <code>haml</code> and <code>erb</code>.

=== Gems used:
* slim
* bootstrap-sass
* font-awesome

=== Bootstrap free template:
* grayscale[http://startbootstrap.com/template-overviews/grayscale/]

=== Slim files found at:
* <code>app/views/home/index.html.slim</code>
* <code>app/views/posts/*.html.slim</code>
