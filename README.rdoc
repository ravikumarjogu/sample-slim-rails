== Sample Rails app with Slim

A sample rails app explaing how slim can be used as template scripting.

Its an alternate to haml and erb.

=== Gems used:
* slim
* bootstrap-sass
* font-awesome

=== Bootstrap free template:
* grayscale

=== Slim files found at:
* app/views/home/index.html.slim
* app/views/posts/*.html.slim
